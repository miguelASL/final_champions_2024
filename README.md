# Análisis de Datos de la Final de la Champions 2024

## Descripción

Este proyecto se centra en el análisis detallado de la final de la champions 2024 entre el **Borussia Dormund** y el **Real Madrid**.

## Requisitos

- Python 3.x
- virtualenv
